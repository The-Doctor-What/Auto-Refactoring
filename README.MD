### Auto Refactoring

Сортировщик файлов в папке по расширениям

### Установка

```sh
yarn install
```

Далее в файле `.env` указать путь к папке, которую необходимо отсортировать

Пример:
```sh
FOLDER=C:\Users\user\Desktop\test
```

### Запуск

```sh
yarn run dev
```

Либо двойной клик на файле `start.bat`
